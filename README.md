# Game Player Gateway Matchmaking

## Construir um **Gateway de jogo** em **TypeScript (Node.js)** que:

- Aceita conexões via **WebSocket**
- Mantém um **lobby de jogadores**
- Permite entrar em uma **fila de matchmaking**
- Garante **regras de consistência**
- Não trava com múltiplos jogadores

Este serviço **NÃO cria partidas**, apenas **orquestra**.
